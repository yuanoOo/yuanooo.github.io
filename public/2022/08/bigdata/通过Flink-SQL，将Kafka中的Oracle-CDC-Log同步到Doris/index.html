通过Flink-SQL，将Kafka中的Oracle-CDC-Log同步到Doris

发表于2022-08-14|更新于2022-08-14|bigdataFlink
|字数总计:811|阅读时长:4分钟|阅读量:

前言

  • Oracle的binlog日志已经由DBA通过OGG同步到Kafka中了，因此用不到Flink CDC
  • 同步到Kafka中的JSON样式

Flink SQL

需要下载以下Jar包，放在{flink_home}/lib/下
flink-sql-connector-kafka_2.12-1.14.5.jar
flink-json-1.15.1.jar
flink-doris-connector-1.14_2.12-1.1.0.jar

  • 开启CheckPoint：SET 'execution.checkpointing.interval' = '10min';

  • 创建Kafka数据源表，设置'format' = 'ogg-json'，只有org.apache.flink.flink-json-1.15.1中以上才支持ogg-json fromat

  • 创建Doris-Sink表

  • 执行INSERT into doris_sink select * from topic_products;语句，写入Doris

Code Repo

  1. bin/sql-client.sh embedded -i init_file -f file -s yarn-session
  2. Execute SQL Files
